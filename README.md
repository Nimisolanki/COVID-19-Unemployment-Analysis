# COVID-19-Unemployment-Analysis
Analyze unemployment trends in India using Python. Perform data cleaning, exploratory data analysis, visualization, and COVID-19 impact analysis to identify unemployment patterns and generate actionable economic and social policy insights.

## How to Open and Run the Notebook

1. Open this GitHub repository.
2. Navigate to the required `.ipynb` notebook file.
3. Click **Open in Colab** (if available). If not, copy the notebook's GitHub URL.
4. Alternatively, visit **https://colab.research.google.com**.
5. Go to **File → Open Notebook → GitHub**.
6. Paste the repository or notebook URL, or search for the repository by name.
7. Select the desired notebook and open it.
8. Once the notebook is loaded, click **Runtime → Run All** to execute every cell in sequence.
